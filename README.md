# 📚 AI in Transcriptomics

## Welcome! 🎉

This is an interactive website supplement for the comprehensive review **"A Review of AI Revolution in Transcriptomics: From Single Cells to Spatial Atlases"**.

---

## Website Content

### Main Pages

- 🏠 **Home** - Overview and Quick Navigation
- 📊 **scRNA-seq Methods** - Detailed list of 70+ methods (Table A)
- 🧬 **Spatial Transcriptomics** - Detailed list of 49 methods (Table B)
- 🚀 **Foundation Models** - 24 foundation models (Table C)
- 🤖 **AI Agents** - 5 AI agents (Table D)
- ℹ️ **More** - Contact information and resources

### Data Included

✅ **70+ scRNA-seq task-specific methods**  
✅ **49 spatial transcriptomics methods**  
✅ **24 foundation models** (parameters, training data, GPU requirements)  
✅ **5 AI agents** (inference cost, access methods)  
✅ **Statistical charts** (bar charts, pie charts, bubble charts)  
✅ **Complete hyperlinks** (GitHub, Web, etc.)
